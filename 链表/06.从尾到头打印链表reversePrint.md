#### 方法三:递归

```
利用递归，先递推至链表末端；回溯时，依次将节点值加入列表，即可实现链表值的倒序输出。

终止条件： 当 head == None 时，代表越过了链表尾节点，则返回空列表；
递推工作： 访问下一节点 head.next ；
回溯阶段： 将当前节点值 head.val 加入列表 tmp ；

void recur(ListNode head){
        if(head == null)
            return;
        recur(head.next);
        list.add(head.val);   
    }
```
